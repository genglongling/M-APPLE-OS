**Algorithm:**

1.  **Initialization:**
    *   Create a list of operations for each job, ordered by step number.
    *   Initialize the start time of each job to 0.
    *   Create a machine availability table, initially setting all machines as available from time 0.

2.  **Scheduling Loop:**
    *   Iterate until all operations are scheduled.
    *   **Priority Rule:**  I will use the "Shortest Processing Time" (SPT) rule.  From the list of unscheduled operations whose predecessors are complete, select the operation with the shortest duration.
    *   **Machine Availability Check:** Check if the required machine is available at the current time. If not, delay the start time of the operation until the machine becomes available.
    *   **Schedule Operation:** Schedule the operation on the machine, update the machine availability table, and mark the operation as completed.
    *   **Update Job Start Times:** If the scheduled operation is the first operation of a job, set the job's start time.

3.  **Conflict Resolution:**
    *   During the machine availability check, if a conflict arises (i.e., another operation is already scheduled on the machine), delay the start time of the current operation until the machine is free.

4.  **Makespan Calculation:**
    *   After all operations are scheduled, the makespan is the maximum completion time of all jobs.

class Operation:
    def __init__(self, job, step, machine, duration):
        self.job = job
        self.step = step
        self.machine = machine
        self.duration = duration
        self.start_time = None
        self.end_time = None

    def __repr__(self):
        return f"Job: {self.job}, Step: {self.step}, Machine: {self.machine}, Start Time: {self.start_time}, End Time: {self.end_time}, Duration: {self.duration}"


def solve_jssp(job_specifications):
    """Solves the Job Shop Scheduling Problem using a priority-rule based heuristic."""

    # 1. Initialization
    jobs = list(job_specifications.keys())
    operations = {}
    machine_availability = {}
    for i in range(20):
        machine_availability[f"Machine{i}"] = 0

    for job_name, job_data in job_specifications.items():
        operations[job_name] = []
        for i, step_data in enumerate(job_data):
            machine = step_data['Machine']
            duration = step_data['Duration']
            operations[job_name].append(Operation(job_name, i + 1, machine, duration))

    scheduled_operations = []
    current_time = 0

    # 2. Scheduling Loop
    while True:
        available_operations = []
        for job_name in jobs:
            # Find the next unscheduled operation for the job
            unscheduled_ops = [op for op in operations[job_name] if op.start_time is None]
            if unscheduled_ops:
                next_op = unscheduled_ops[0]

                # Check if all preceding operations are complete
                preceding_ops_complete = True
                if next_op.step > 1:
                    for i in range(next_op.step - 1):
                        if operations[job_name][i].end_time is None:
                            preceding_ops_complete = False
                            break

                if preceding_ops_complete:
                    available_operations.append(next_op)

        if not available_operations:
            break  # All operations scheduled

        # 3. Priority Rule (SPT - Shortest Processing Time)
        if available_operations:
            shortest_op = min(available_operations, key=lambda op: op.duration)
        else:
            break

        # 4. Machine Availability Check and Conflict Resolution
        machine = shortest_op.machine
        start_time = max(current_time, machine_availability[machine])

        # Schedule the operation
        shortest_op.start_time = start_time
        shortest_op.end_time = start_time + shortest_op.duration
        machine_availability[machine] = shortest_op.end_time
        scheduled_operations.append(shortest_op)

        # Update current time (not strictly necessary for this SPT implementation, but good practice)
        current_time = min([machine_availability[m] for m in machine_availability])

    # 5. Makespan Calculation
    makespan = max(op.end_time for op in scheduled_operations)

    return makespan, scheduled_operations
