def solve_jssp(jobs):

    Solves the Job Shop Scheduling Problem using a heuristic approach.

    Args:
        jobs: A list of jobs, where each job is a list of operations.
              Each operation is a tuple: (machine, duration).

    Returns:
        A tuple: (makespan, schedule).
