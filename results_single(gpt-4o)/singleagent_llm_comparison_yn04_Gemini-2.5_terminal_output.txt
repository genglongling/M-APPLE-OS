**Algorithm:**

1.  **Initialization:**
    *   Create a list of operations for each job, ordered by step number.
    *   Initialize the start time of each job to 0.
    *   Create a machine availability table, initially setting all machines as available from time 0.

2.  **Scheduling Loop:**
    *   Iterate until all operations are scheduled.
    *   **Priority Rule:**  I will use the "Shortest Processing Time" (SPT) rule.  From the list of unscheduled operations whose predecessors are complete, select the operation with the shortest duration.
    *   **Machine Availability Check:** Check if the required machine is available at the current time. If not, delay the start time of the operation until the machine becomes available.
    *   **Schedule Operation:** Schedule the operation on the machine, update the machine availability table, and mark the operation as completed.
    *   **Update Job Start Times:** If the scheduled operation is the first operation of a job, set the job's start time.

3.  **Conflict Resolution:**
    *   During the machine availability check, if a conflict arises (i.e., another operation is already scheduled on the machine), delay the start time of the current operation until the machine is free.

4.  **Makespan Calculation:**
    *   After all operations are scheduled, the makespan is the maximum completion time of all jobs.

    def __repr__(self):
