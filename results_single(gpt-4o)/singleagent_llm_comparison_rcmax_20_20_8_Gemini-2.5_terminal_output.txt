    *   Initialize the start time of each job to 0.

    *   **Schedule Operation:**
        *   Find the earliest available time slot on the required machine that is at least as long as the operation's duration and after the job's current start time.

        *   Update the machine availability table.
        *   Update the job's start time to the end time of the scheduled operation.
        *   Mark the operation as completed.
